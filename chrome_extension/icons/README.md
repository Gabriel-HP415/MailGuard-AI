# MailGuard-AI — Chrome Extension icons

You need three PNG icons to ship the extension:

| Size | Use |
|------|-----|
| 16x16 | Toolbar (small) |
| 48x48 | Extension management |
| 128x128 | Install / store listing |

Generate them from a single 512x512 master and save as:

- `icon16.png`
- `icon48.png`
- `icon128.png`

A simple shield logo with a gradient blue/violet is recommended. Until real
icons are added, Chrome will use a placeholder.
